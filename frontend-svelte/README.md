# Smart TODO — Svelte

Фронтенд на Svelte для дипломного проекта «Создание и сравнение трёх реализаций TODO-приложения на разных фреймворках».

## Технологии

- Svelte 5
- Vite
- svelte-spa-router
- Чистый JavaScript, чистый CSS
- Оранжевая цветовая схема

## Запуск

```bash
npm install
npm run dev
```

Приложение: http://localhost:5173

## Переменные окружения

- `VITE_API_URL` — URL бэкенда (по умолчанию http://localhost:8000)

## Функционал

- Регистрация / авторизация
- Проекты (CRUD)
- Задачи (CRUD, роли)
- Категории (CRUD)
- Дашборд и статистика
- Чат задачи (WebSocket)
- Общий чат (WebSocket)
